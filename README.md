# Bài tập Dart

Dự án lưu các bài tập Dart cơ bản chạy trên console.

## Bài tập hiện có

- Bài 1 - Tính lương nhân viên: `bin/main.dart`
- Bài 2 - Quản lý đơn bán hàng: `bin/bai_02_quan_ly_don_hang.dart`

## Cách chạy

```bash
dart pub get
dart run bin/main.dart
dart run bin/bai_02_quan_ly_don_hang.dart
```
